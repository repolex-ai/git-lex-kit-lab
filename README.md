# git-lex-kit-lab

Collaborative research kit for [git-lex](https://github.com/repolex-ai/git-lex).

Git-native research methodology where branches embody hypotheses and commits are evidence.

## Document Types

- **Investigation** — research project container
- **Hypothesis** — testable claims
- **Experiment** — test procedures and results
- **Survey** — landscape analysis
- **Synthesis** — conclusions drawn from evidence
- **Deliberation** — structured group reasoning
- **ResearchBranch** — git branch as research thread

## Install

```bash
git lex init --kit lex-lab
```
